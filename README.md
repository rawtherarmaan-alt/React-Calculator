# React Calculator

A simple calculator application built using React and React Hooks, demonstrating 
core frontend concepts like state management and event handling.

## Features
- Perform basic arithmetic operations: addition, subtraction, multiplication, division
- Clean, responsive UI
- Real-time input and result display

## Tech Stack
- **Frontend:** React.js
- **State Management:** React Hooks (useState)
- **Build Tool:** Vite

## What I Learned
- Managing component state using the useState hook
- Handling user input and button click events in React
- Structuring a small React application with reusable components
- Using Vite for fast development and build setup

## How to Run
```bash
npm install
npm run dev
```

Then open the local server link shown in your terminal.
